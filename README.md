# Carrera de Barras de Progreso - Actividad 07

## Descripción
Proyecto desarrollado para la Actividad 07 sobre el manejo de Hilos y concurrencia en C#.
Consiste en una aplicación de Windows Forms que simula una carrera entre tres barras de progreso que avanzan de forma independiente y aleatoria.

## Estructura
El proyecto se organiza en los siguientes archivos principales:

Form1.cs: Lógica de los hilos, control de la carrera y actualización de la interfaz.
Form1.Designer.cs: Definición de controles (ProgressBars, Labels, Button y ListBox).
Program.cs: Punto de entrada que inicia la ejecución del formulario.

## Requisitos
.NET 6.0 o superior (según la configuración del proyecto).
Visual Studio 2022 o el SDK de .NET instalado.

## Comandos
```bash
# Restaurar dependencias
dotnet restore

# Compilar el proyecto
dotnet build

# Ejecutar la aplicación
dotnet run --project CarreraBarras.csproj
```
